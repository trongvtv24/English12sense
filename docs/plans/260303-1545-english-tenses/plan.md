# Plan: Ứng dụng Học 12 Thì Tiếng Anh Cho Bé (English Tenses App)
Created: 03/03/2026
Status: 🟡 In Progress

## Overview
Xây dựng Web App tương tác giúp bé 9 tuổi học 12 thì tiếng Anh qua đồ họa Timeline sinh động và 6 mini-games luyện phản xạ. Sử dụng Next.js làm Frontend và Supabase quản lý Database.

## Tech Stack
- Frontend: Next.js + React + TypeScript
- Animation: Framer Motion
- Backend/Database: Supabase
- Deployment: Vercel
