# Phase 01: Setup Environment
Status: ⬜ Pending | 🟡 In Progress | ✅ Complete
Dependencies: None

## Objective
Khởi tạo dự án Next.js, cài đặt các thư viện thiết yếu (Framer Motion, Supabase UI, Tailwind) và chuẩn bị môi trường code.

## Implementation Steps
1. [ ] Khởi tạo dự án Next.js với TypeScript/Tailwind: `npx create-next-app@latest .`
2. [ ] Cài đặt lib animation và kéo thả: `npm i framer-motion @hello-pangea/dnd`
3. [ ] Cài đặt Supabase Client: `npm install @supabase/supabase-js @supabase/ssr`
4. [ ] Cài đặt icon/utils: `npm i lucide-react clsx tailwind-merge`
5. [ ] Cấu hình `.env.local` bổ sung keys Supabase.
6. [ ] Tạo folder structure (`src/components`, `src/app`, `src/lib`, `src/hooks`).
